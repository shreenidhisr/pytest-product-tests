# pytest-product-tests

Companion test repo for `java-product-api`. Contains hand-written system tests and AI-generated tests produced by the CI workflow.

---

## Running Tests

```bash
# Install dependencies
pip install -r requirements.txt

# Start the Java API first (in the java-product-api repo)
# ./mvnw spring-boot:run

# Run all tests
pytest tests/ -v

# Run with HTML report
pytest tests/ -v --html=report.html --self-contained-html
```

By default, tests hit `http://localhost:8080`. Override with:

```bash
BASE_URL=http://my-server:8080 pytest tests/ -v
```

## AI-Generated Tests

Tests with filenames like `test_description_field.py` or `test_ai_pr_42.py` are auto-generated by the `java-product-api` CI workflow using Claude. Always review them before merging the PR.
